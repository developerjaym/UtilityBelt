Metadata

Provide a title, subtitle, and tags. On the homepage, you can search by title and tags.

Inputs

Add all the inputs you need. Give each input a label, a type (like date or number or color), and optionally a default value. These inputs will be used to generate your custom form.

Function

This is where you write the JavaScript function body. The function has five arguments. The first, paramArray, is an array that holds the values of each input from your custom form. The second, log, is a function that outputs to the right side of the output pane. The third, print, is a function that outputs to the left side of the output pane. Use this function to output your results. The fourth, localStorage, is a proxy to localStorage. Use it the same way. The fifth, CORS_URL, provides a proxy that prevents CORS issues. Just do something like fetch(CORS_URL + "https://example.com").
