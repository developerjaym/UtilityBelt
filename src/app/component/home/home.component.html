Overview

Search through, run, create, and share forms and functions.

Example Problem

I often need to generate a sample work order JSON payload.

How to Solve with UtilityBelt

I hit 'CREATE'. I give my custom form a name, subtitle, and tags (for searching). I then define the inputs for my custom form: customer number, work order type, and priority. I then write the function that takes those three inputs' values and turns them into a valid JSON payload. I then test my form and function and hit 'SAVE' when it works. My custom form is now in localStorage, and I can easily search for it in the future.
